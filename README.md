# Sjakkmatte

Her er materiale til *Sjakkmatte*, som inneholder morsomt undervisningsopplegg som kobler sjakk og matematikk. Laget for ungdomsskolen, men kan kanskje brukes på andre trinn.

Opplegg med bruk av programmering. Følgende linker sender deg til en interaktiv plattform som kan brukes av både lærere og elever. Det kan ta noe tid å laste inn siden.
